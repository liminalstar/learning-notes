### 基本① 型ってなんだ？

結論から言います。
> 型とは、データの種類をことを言います。

データの種類には以下のような物があります。

| 型名           | 例                           | 説明                 |
| ------------ | --------------------------- | ------------------ |
| `int`        | `1`, `-3`, `0`              | 整数                 |
| `float`      | `3.14`, `-0.01`             | 小数（浮動小数点）          |
| `complex`    | `1 + 2j`                    | 複素数                |
| `bool`       | `True`, `False`             | 真偽値型（`int` のサブクラス） |
| `str`        | `"hello"`, `'文字列'`          | 文字列                |
| `list`       | `[1, 2, 3]`, `["a", "b"]`   | 可変リスト（順序あり・重複可）    |
| `tuple`      | `(1, 2, 3)`, `("a", "b")`   | 不変リスト（順序あり・変更不可）   |
| `set`        | `{1, 2, 3}`, `{"a", "b"}`   | 集合（重複なし・順序なし）      |
| `dict`       | `{"a": 1, "b": 2}`          | 辞書（キーと値のペア）        |
| `NoneType`   | `None`                      | 値がないことを示す          |
| `bytes`      | `b"abc"`                    | バイナリ文字列（変更不可）      |
| `bytearray`  | `bytearray(b"abc")`         | バイナリ文字列（変更可）       |
| `range`      | `range(0, 5)`               | 整数の範囲オブジェクト        |
| `memoryview` | `memoryview(b"abc")`        | バッファアクセス用のオブジェクト   |
| `function`   | `def f(): pass`             | 関数オブジェクト           |
| `type`       | `type(1)` → `<class 'int'>` | 型オブジェクト（型そのもの）     |
| `object`     | `object()`                  | すべての型のベース（基底クラス）   |

沢山出てきて混乱してますね？
大丈夫です。だんだん覚えていきましょう。

### 基本③ 変数名と型はセット

基本①と②で、型と変数についてざっくり説明しました。

- **変数**は、場合により値が変わる文字。 xとかaとか

- **型**は、データの種類。 文字とか数値とか

なぜこれらがセットなのか？結論から言います。

> 変数にどんなデータ種類のデータを入れたかわかるようにするため

です。

人もわかりやすいというのがありますが、本質はコンピュータが解釈できるというところにあります。

例えば、以下の数字があったとします。

```c
    // c言語が分かりやすいので、ｃで説明しています。
    int c = 65;  // cという変数の型がint:整数型なので65と認識されます。
    char x = 65; // xという変数の型がchar:文字型なので対応する文字、Aと認識されます。
```

上記のように値は同じだが、型が違うだけコンピュータの認識は全然変わってきます。

取り扱ってほしいデータの種類をしっかりコンピュータに認識してもらうために型というデータの種類を変数に設定するわけですね。

> ただPythonは、そんな型設定を自動でやってくれます！

覚える意味ないじゃんって思ったあなた。知っておくとエラー対応で役に立ちますよ。

## 第4章 基本構文を知ろう

また、基本です。基本が肝心です。

基本構文というものが必ずあります。言語が違っても書き方が変わるだけで本質は変わりません。

Pythonの基本構文を以下に一覧で出します。

| 内容        | 構文例                                           | 説明               |
| --------- | --------------------------------------------- | ------------------------------ |
| **変数代入**  | `x = 10`                                      | 値を変数に保存する            |
| **関数定義**  | `def greet(name):`<br>` print("Hello", name)` | 処理をまとめて呼び出せるようにする |
| **条件分岐**  | `if x > 0:`<br>` print("正")`                  | 条件に応じて処理を分ける          |
| **繰り返し**  | `for i in range(3):`<br>` print(i)`           | 繰り返し処理                   |
| **コメント**  | `# これはコメントです`                                 | プログラムの説明やメモ   |
| **インデント** | `  print(x)`              | Pythonではインデント（字下げ）が意味を持つ（ブロック） |
| **出力**    | `print("Hello")`                              | 画面に出力する               |
| **入力**    | `input("名前は？")`                               | ユーザーから入力を受け取る       |

これらが基本中の基本です！

主要な部分をそれぞれ個別で説明していきます。

### 変数代入

> プログラミングは、右辺から左辺に値を代入するというルールで成り立っています。

逆はありません。左辺から右辺は、算数・数学の話ですね！なので、、

```python
    # python
    x = 10 # xに10を代入します。
    a = "hello world" # aにhello world という文字列を代入します。
```

はじめはなれないかもしれませんが合言葉は、「右辺から左辺へ代入」と覚えてください。

### 出力

> 画面に結果を出力するときにつかいます。コマンドラインという場所(黒画面)に結果が出てきます。

```python
    # python
    print("以下、変数a出力↓")
    print(a) # ()の中に変数や文字列を入れるとその値を出力してくれます。
```

プログラムの確認で使うことが多いですね！

### 入力

> ユーザーからの入力を受け取る関数です。

```python
    # python
    # input関数は、ユーザーから入力を受取り変数mojiに代入する。
    moji = input("好きな入力をしてください：") 
    print(moji)
```

※ input関数から取得されるデータの種類(型)は、文字列になります。

-> ユーザーがキーボードで数字の0を入力しても文字の"0"と認識します。

### 条件分岐

> 条件に応じて処理を分岐させる関数です。通称if文とも言われます。

```python
    # python
    no = 0
    if no == 1: # もしno変数の値が1だったら、、
        print("if条件が正の場合にこちらが処理されます。")
    elif no == 2: # もしno変数の値が2だったら、、
        print("elif条件が正の場合にこちらが処理されます。")
    else: # もし既定の条件になにもがいとうしなかったら、、
        print("全条件が負の場合にこちらが処理されます。")
```

※ vbとかをやられている方は、switch文とかあると思いますがpythonにはそれがないので基本ifで対応します。

また、elifが大量にあるのは読みやすいコードとしては適切でないのでelifの追加はほどほどにしましょう。

上記以外にも比較するための書き方がありますので表を乗っけます。

> 比較演算子

| 演算子  | 意味    | 例        | 結果      |
| ---- | ----- | -------- | ------- |
| `==` | 等しい   | `5 == 5` | `True`  |
| `!=` | 等しくない | `5 != 3` | `True`  |
| `>`  | より大きい | `5 > 3`  | `True`  |
| `<`  | より小さい | `5 < 3`  | `False` |
| `>=` | 以上    | `5 >= 5` | `True`  |
| `<=` | 以下    | `3 <= 5` | `True`  |

> 論理演算子

| 演算子   | 意味          | 例                  | 結果               |
| ----- | ----------- | ------------------ | ---------------- |
| `and` | かつ（両方が真）    | `x > 0 and x < 10` | `True`（xが1〜9なら）  |
| `or`  | または（どちらかが真） | `x < 0 or x > 10`  | `True`（xが範囲外なら）  |
| `not` | 否定（真偽を反転）   | `not x == 0`       | `True`（xが0でないなら） |

組み合わせて、自分のオリジナルの条件分岐を作りましょう！

### 繰り返し

> 繰り返し処理する時の構文です。通称for文と言います。

```python
    # python
    for i in range(10): # range()は10回 forの中の処理を繰り返すという意味です。
        print(i) # iを出力します。10回繰り返すが値は0~9が出力される
```

0~9が出力される理由は、カウントの始まりが0からだからという理由です。

for の中にforを入れるなど2重for文のような書き方がありますが、やりすぎには注意しましょう。

### 関数定義

> オリジナルの関数を定義できます。通称ユーザー定義関数ともいいます。

中学校の時、こんな物を学習したことを覚えていますか？

> f(x) = 3x + 2

これは関数fにxを入れたときの値を表している式でしたね。
fが関数名で、入れる値x(引数とも呼びます)で構成された式になります。

この考え方は度のプログラミング言語でもそのまま適用されます。

Pythonで書くとこうなります。

```python
    # python
    def f(x):
        return 3 * x + 2
    
    result = f(x=1)
    print(result)
```

読み方は同じです。fという関数にxの値を入れた時の値が戻ってきます。
今回は、xに1を入れてますのでresult の値は、5になって帰ってきます。(return)

使いまわすようなスクリプト(プログラムの塊)は、関数でまとめたほうがすっきりするので使用を考えてください。

もちろん、複数の値を入れることもできます。
